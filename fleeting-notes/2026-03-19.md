# 2026-03-19 Fleeting Notes

## 21:52 - CSS Customizable Selects

Interesting CSS-Tricks article on the new `appearance: base-select` feature:

- New pseudo-elements: `::picker(select)`, `::picker-icon`, `::checkmark`
- `sibling-index()` and `sibling-count()` functions for position-aware styling
- Works with anchor positioning for dropdown placement
- `@starting-style` for entry animations on appearing elements
- Progressive enhancement: non-supporting browsers just show normal `<select>`

Creative demos:
1. Curved folder stack (progressive rotation)
2. Card deck fan (spread animation)
3. Radial emoji picker (cos/sin positioning)

Link: https://css-tricks.com/abusing-customizable-selects/

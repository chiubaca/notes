## 05:51 UTC

Exploring [Alchemy v2](https://v2.alchemy.run/) — TypeScript IaC on Effect. Cloudflare infra (Workers, R2, KV, D1) defined as TypeScript code with `Effect.gen`. Could be my gateway into learning Effect/ts properly.

- Async mode available (no Effect required) — Alchemy infers fully typed `env` from bindings without codegen
- `alchemy dev` does full-stack hot reload against live services (~100ms)
- `alchemy test` runs isolated integration tests on real cloud resources — no mocks
- Supports Vite, Astro, TanStack, SolidStart, Nuxt, Vue

Goal: spin up a Cloudflare Worker + R2 stack as a first project, get comfortable with Effect patterns in the process.

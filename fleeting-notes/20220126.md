- github co-pilot taught me a stonker of css trick today:

Add ellipsis to truncate long text after x amount of lines:

```css
  overflow: hidden;
  text-overflow: ellipsis;
  display: -webkit-box;
  -webkit-line-clamp: 2;
  -webkit-box-orient: vertical;
```
